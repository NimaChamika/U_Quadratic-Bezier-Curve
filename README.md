# U_Quadratic-Bezier-Curve

Quadratic Bezier Curve implemented using "De Casteljau's algorithm".

For a quadratic bezier curve we need 3 points. Then the formula is P = (1−t)2P1 + 2(1−t)tP2 + t2P3
